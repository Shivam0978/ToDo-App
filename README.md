#  Todo App

A modern and responsive Todo App designed to help users organize and manage daily tasks efficiently. Users can add, edit, complete, and delete tasks through a clean and intuitive interface. Built with a focus on simplicity, productivity, and a seamless user experience across desktop and mobile devices.

##  Features

*  Add new tasks
*  Edit existing tasks
*  Delete tasks
*  Fully responsive design
*  Clean and user-friendly interface

##  Installation

1. Clone the repository

```bash
git clone https://shivam0978.github.io/ToDo-App/
```

2. Navigate to the project folder

```bash
cd todo-app
```

3. Open `index.html` in your browser.

##  Usage

* Enter a task in the input field.
* Click the Add button to create a new task.
* Edit tasks whenever needed.
* Mark tasks as completed to track progress.
* Delete tasks you no longer need.
